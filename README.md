# README
## 開発言語
- Ruby 3.0.1
- Rails 6.1.7.6
## 実行手順
~~~
$ git clone git@github.com:Mottttton/yaeh_map.git
$ cd yaeh_map
$ bundle install
$ yarn install
$ rails db:create && rails db:migrate
$ rails s
~~~
